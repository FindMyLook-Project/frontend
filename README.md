# React + Vite

FindMyLook - Frontend (React + Vite)
FindMyLook frontend infrastructure built with React, Vite, and Tailwind CSS 4.

Prerequisites
Node.js (Version 20.19.0 or higher)

Git installed and configured

Installation and Setup
Follow these steps to set up the project on your local machine:

1. Clone the Repository
Bash
git clone https://github.com/FindMyLook-Project/frontend.git
cd frontend
2. Install Dependencies
This command installs all required packages including Tailwind CSS 4, React Router, and Axios:

Bash
npm install

3. Run Development Server
Bash
npm run dev
The application will be available at the URL shown in your terminal (typically http://localhost:5173).
